## Getting Started
### Prerequisites
Before starting the project, make sure you have these installed :
- Node & NPM
- PostgreSQL

### Installation

Clone the repo :
```
cd ./the/path/of/your/choice
git cloclone git@github.com:theoprovost/theoprovost.fr.git
```
Launch the project :
```
cd ./repo
npm install
node index.js
```
You can also use `nodemon` :
```
nodemon index.js
```
